HOLLOWCRAFT — how to play co-op with a friend
==============================================

Nothing to install. Node is already bundled in this folder.

STEP 1 — Same network?
  - Same house / same WiFi:  skip to step 2.
  - Over the internet:  both of you install Hamachi (free) and join the
    SAME Hamachi network. https://vpn.net  (one of you makes a network,
    the other joins it with the network ID + password). That makes your
    PCs act like they're on the same LAN.

STEP 2 — Launch the game
  - Double-click  Hollowcraft-LAN.bat
  - A black window opens (leave it open) and the game opens in your browser.
  - If Windows Firewall asks, click ALLOW (on Private AND Public if shown).

STEP 3 — One of you hosts
  - Menu > Multiplayer > "Host a game".  You're now the host.

STEP 4 — The other joins
  - Menu > Multiplayer.  The host's game appears in the LAN list.
  - Click "Join".  You're in.

That's it. If the list is empty, double-check you're both on the same
Hamachi network and the host clicked "Host a game" first.

Tip: the host can also just send you their link  http://<their-hamachi-ip>:8788
and you open it in a browser — you'll see their game as "(this server)".
